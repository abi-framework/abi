Are you eager to start a TypeScript-based web project? Abi is the perfect solution to serve your resources, web services or applications in no time and without any hassle!

:::caution[Work in progress]
[Abi](https//github.com/siguici/abi) is in early development.
If you find something that’s not working, [open an issue on the GitHub repository](https://github.com/abi-js/abi/issues/new/choose).
:::

## 📋 Requirements

Currently, `Abi` only works under:

  > [Bun](https://bun.sh/docs/api/http#object-syntax),
  > [Deno](https://docs.deno.com/runtime/manual/tools/serve/),
  > [Cloudflare Workers](https://developers.cloudflare.com/workers/runtime-apis/fetch/)
